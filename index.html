Introducing the advanced Blind Stick: a smart navigation aid for the visually impaired. This innovative stick utilizes ultrasonic sensors to detect obstacles, LDR to sense lighting conditions, and an RF remote for remote locating. Real-time feedback is provided through a buzzer or vibrator motor, enhancing the user's navigation experience. Experience improved independence and ease with our technologically advanced Blind Stick.

Smart LPG Monitoring System

A system was developed to measure gasoline levels in a cylinder using an RFID pressure sensor. The sensor's output was fed to a PIC controller, which stored the voltage representing the gas weight. A threshold value was set in the controller. The LCD displayed the value, and an alarm triggered when a set threshold was reached.
